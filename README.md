# AI PO Assistant
A cloud-hosted AI assistant with upgraded UI, designed for project managers and product owners.

## Features
- Clean and futuristic UI
- Conversation history
- Free cloud hosting
- No weird characters

## Deployment
1. Push repo to GitHub.
2. Go to [Streamlit Cloud](https://share.streamlit.io).
3. Click **New App → Select Repo**.
